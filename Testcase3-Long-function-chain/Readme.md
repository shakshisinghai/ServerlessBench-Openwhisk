## How to Deploy
```
cd ./Sequence-nested/action/
Run ./action_update.sh in the ./action/
```

## How to Invoke
```
cd ./Sequence-nested/action/
Run ./action_invoke.sh or ./action_invoke_seq.sh in the ./action/
```
